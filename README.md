# Sales-Data-Analysis-using-SQL
Sales Data Analysis using SQL with advanced queries and business insights

# Project Overview
This project focuses on analyzing sales data using SQL to extract meaningful business insights. The dataset contains information about orders, customers, products, and regions.

The objective of this project is to perform data analysis using SQL queries and answer real-world business questions.

# Tools Used
- MySQL
- SQL (Window Functions, Aggregations, Subqueries)

# Key Analysis Performed
- Total Revenue and Order Analysis
- Region-wise Sales Performance
- Top Selling Products
- Customer Purchase Behavior
- Monthly Sales Trends
- Advanced SQL Analysis using Window Functions

# Advanced Concepts Used
- RANK() and ROW_NUMBER()
- PARTITION BY
- Subqueries
- CASE Statements
- Aggregate Functions

# Key Insights
- Identified top-performing products and regions
- Analyzed customer purchase patterns
- Detected high-value customers
- Found seasonal sales trends

# Project Structure
- queries.sql → Contains all SQL queries used in analysis

# Conclusion
This project demonstrates strong SQL skills and ability to derive business insights from raw data.
